# Task 3 - Restaurant Menu UI

This project is part of my UI/UX Internship at CodSoft.

## 📌 Project Overview
Designed a Restaurant Menu UI for "CAFENOVA", focusing on creating a clean and user-friendly food browsing experience.

## 🎯 Features
- Categorized menu (Starters, Main Course, Beverages, Desserts)
- Multiple item variants with pricing
- Interactive "Add" system with cart preview
- Consistent dark-themed UI
- Focus on usability and readability
